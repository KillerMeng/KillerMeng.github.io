RxJava

发表于 2016-03-08   |   分类于 源码   |  

RxJava,一个大家最近吵的很火热的词.

RxJava is a Java VM implementation of ReactiveX (Reactive Extensions): a library for composing asynchronous and event-based programs by using observable sequences.

大约就是这个意思:RxJava是一个通过观察队列实现的异步的,基于事件的在JVM的库

关于ReactiveX

Rx = Observables + LINQ + Schedulers。

使用观察者模式
  • 创建：Rx可以方便的创建事件流和数据流
  • 组合：Rx使用查询式的操作符组合和变换数据流
  • 监听：Rx可以订阅任何可观察的数据流并执行操作
简化代码
  • 函数式风格：对可观察数据流使用无副作用的输入输出函数，避免了程序里错综复杂的状态 简化代码：Rx的操作符通通常可以将复杂的难题简化为很少的几行代码
  • 异步错误处理：传统的try/catch没办法处理异步计算，Rx提供了合适的错误处理机制
  • 轻松使用并发：Rx的Observables和Schedulers让开发者可以摆脱底层的线程同步和各种并发问题

概念 扩展的观察者模式

观察者模式就不用多说了,大家都懂的,android中最常见的就是view.setOnClickListener().不过这里的观察者,通过注册(register)或者订阅(Subscribe)的方式实现.
RxJava 的观察者模式有是个基本的状态

  • Observable(可观察者)
  • Observer(观察者)
  • subscribe(订阅)
  • 事件

不同意传统的观察者模式,onNext()[onClick()/onEvent()]这里增加两个:

  • onCompleted()
  • onError()

onCompleted():事件队列完成,RxJava不仅把每个事件都单独处理,还把他们当成一个队列处理,当不再有新的onNext()出现的时候,就得调用onCompleted()

onError():事件队列异常,onError会触发,同时队列自动终止,不允许再有事件发出

onCompleted()和onError()是互斥的.

实现

  1. 创建Observer(观察者)
  2. 创建Observable(被观察者)
  3. 创建Subscribe(订阅)

关于梦想,关于现实

发表于 2016-02-29   |   分类于 生活   |  

一个醉酒的不眠夜,想和自己说说话.

当初开这个博客的时候,是想以技术博客为主的.然而两个月过去了,发现自己还是当初的自己,每天还是习惯在笔记本上写写花花,而这里嫣然成了一个我记小技巧笔记的地方.

而我今天更想和自己说说话,在这个安静的地方.

他们说今天是2月29,他们说想看到四年以后一个更好的自己,他们说了太多太多,或矫情,或BB,或文艺,或不着边际.

我什么也没说,其实想祝两个好朋友生日快乐.

其中的一个过生日的好朋友,马上也要去大帝都了,恍惚间发现周围已经走的没人了,他们也在调侃我,让我去大帝都,去大魔都,其实我内心根本就不想去.不过还是衷心的祝福他们,毕竟舞台,还是大点好.晚上匆匆忙忙的吃了个火锅,晕晕乎乎的喝了点啤酒,然后又求迷杏眼的喝了杯咖啡,然后又匆匆忙忙的回家.

说说其他的吧,如果在家吃饭,妈妈会给我夹吃的,说她不爱吃,然我吃.我会不知道是妈妈真的不爱吃,还是不舍得,想让我吃.朋友,兄弟相处之道也是如此,很多时候,我不知道朋友做的事情,是为了我,还是出于自己的本心.虽然知道人与人相处就是这样,但是还是很奢求能有一两个可以说真心话,去掉大脑过滤说话的朋友,不过好像不可能遇到.

再说说年龄,这几天,面试了很多工程师,他们其中有很多人,有多年的工作经验,虽然水平不敢恭维,但是还是希望很高的薪资,就是这样,生活不易,且加油且奋斗吧.

再说说自己吧,经常会调侃,经常会瞎扯,会说自己想娶2个媳妇,想啪啪啪黑人,想要皮裤妹子.其实如果真心让我说,我也不知道我想要什么,或许,只是内心的怦然心动,加上大脑短路的决定,加上赌场赌徒的心态吧.女孩,曾几何时,已经成了奢侈品.自己一个人,回窃喜,会狂欢.

在说说创业,从当初的雄心壮志,到现在的且行且珍惜,在路上,就继续吧,终究是要走的,仅此而已.父母的养育,兄弟的付出,女人的陪伴,都是辜负不起的,如果输,就输到低吧.

再说说生活,每天写代码,听音乐,看天空,就是这样,房东的功放,自己败的圈铁耳机,还有launchpad.我觉得挺好的,至少,我还能快乐起来.

再说说计划,手头的任务甩不完,接手的工程师招不到,明天开始,尝试下双倍工作吧,把app和游戏同时启动,试试看.

眼睛不舒服,去睡觉了

这是我的博客的第一篇文章.

很久之前就有写博客的想法了,不过总是因为各种原因没有去做。记得大约在去年，还在CSDN和开源中国之间徘徊过,后来手机用简书的时候,也考虑过在简书上开一个自己的博客,可惜这些想法最后都没有结果。
最后之所以准备开这个博客，一方面是好朋友给我推荐了Github+Hexo，二来是静态博客最近确实很火。加上新的一年到了，也准备给自己定点计划和目标，所以才强迫自己开了这个博客。

至于www.3toptop.top这个域名，本来是想买个自己喜欢的词缀的顶级域名的，可惜都被抢注了，然后脑洞打开，就选了这个域名…然后还起了这个很中二的名字（做数据统计，域名解析的时候，都是叫3-Top），然后就因为当时的主题的样式的标题上下有两道横线，然后感觉不喜欢，就被抛弃了。。。。。


虽然俺是程序猿，但程序猿也是有生活的好吗？加上这个博客应该是不会被搜索引擎收到的，所以我想记录的东西都可以放到这里，技术分享，学习笔记，生活经历，逗比玩法什么的，都会尝试去写。
